# Room - SleepQualityTracker app

## SleepQualityTracker
수면에 대한 정보를 기록하는데 도움을 주는 토이 앱
- 시작 시간 
- 종료 시간
- 품질
- 수면 시간
을 기록합니다.


## Techniques
* Room database
* DAO
* Coroutines
* Transformation map
* Data Binding in XML files
* ViewModel Factory
* Using Backing Properties to protect MutableLiveData
* Observable state LiveData variables to trigger navigation
